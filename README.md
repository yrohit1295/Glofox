# Glofox
The project is built in Laravel framework.

## Requirements
Please install mentioned below dependencies
```
Install Apache2
Install PHP
Install mysql
```
After installation, please run database.sql file to create database and tables.

## API's:
```
/api/classes
/api/bookings
```

## Run test cases:
```
cd /var/www/html/Glofox
./vendor/bin/phpunit
```
